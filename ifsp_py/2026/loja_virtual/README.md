🛒 Sistema de E-commerce em Python

Este projeto é uma simulação simples de um sistema de e-commerce desenvolvido em Python, com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO). O sistema representa o fluxo básico de uma compra online, incluindo cadastro de cliente, gerenciamento de carrinho de compras, escolha de método de pagamento e tipo de entrega.

Durante o desenvolvimento foram utilizados conceitos importantes como herança, encapsulamento, classes abstratas e organização do código em módulos, permitindo uma estrutura mais clara e reutilizável.

O sistema permite criar produtos, adicioná-los ao carrinho com diferentes quantidades, remover itens, calcular o valor total da compra e finalizar o pagamento utilizando diferentes métodos, além de incluir o cálculo do frete conforme o tipo de entrega escolhido.

▶️ Como executar

Clone o repositório:
git clone <URL_DO_REPOSITÓRIO>

Entre na pasta do projeto:
loja_virtual

Execute o programa em:
python main.py
